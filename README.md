# Zen Garden Interactive

Вебдодаток для створення цифрового японського саду каменів. Проєкт реалізований на HTML5 Canvas і дозволяє малювати візерунки на піску, розміщувати каміння та дерева, а також слухати фонову музику.

## Функціональні можливості
- Малювання ліній граблями з ефектом об'єму.
- Автоматичне виявлення перешкод: лінії не перекривають встановлені камені та дерева.
- Налаштування розміру пензля та елементів через панель керування.
- Система очищення полотна та інтерактивний аудіоплеєр.

## Технічний стек
- HTML5 (структура)
- CSS3 (дизайн, Glassmorphism)
- Vanilla JavaScript (логіка малювання, фізика зіткнень)

## Встановлення та запуск
1. Клонуйте репозиторій:
   git clone https://github.com/username/zen-garden.git
2. Відкрийте index.html у будь-якому сучасному браузері.

## Структура проєкту
- index.html: основна розмітка.
- style.css: стилізація інтерфейсу та полотна.
- script.js: вся логіка взаємодії та візуалізації.
- assets/: аудіофайли та інші ресурси.
